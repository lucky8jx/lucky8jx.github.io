使用 vuepress 搭建简单博客

如何使用 vuepress 搭建一个简单的博客呢？

  • 使用 vuepress 初始化项目

  • 配置 mini 主题

# 创建并进入一个新目录

  mkdir vuepress-starter && cd vuepress-starter

# 使用包管理器进行初始化

npm init -y

输出结果如下：

# 将 vuepress 安装在本地

npm i -D vuepress

# 创建第一篇文档

mkdir docs && echo '# Hello VuePress' > docs/README.md

# 在 package.json 中添加一些 scripts

{
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}

# 在本地启动服务器

npm run docs:dev

运行输出结果如下：

浏览器打开 localhost:8080 显示如下：

# 安装 vuepress-theme-mini

npm i vuepress-theme-mini -D

# 添加主题配置

// docs/.vuepress/config.js
module.exports = {
  title: '简单博客',
  discription: '通过 vuepress 搭建的简单博客',
  theme: 'mini',
  themeConfig: {
    siteName: '我的博客',
    smoothScroll: false,
    lastUpdated: 'Last Updated',
    author: 'lucky8jx',
    navbar: true,
    nav: [
      { text: '首页', link: '/' },
      { text: '归档', link: '/archive/' },
    ],
  }
}

# 修改之前创建的第一篇文档 docs/README.md

---
layout: Pager
---
