# Java变量

一个变量就是一个能够存储变化的值的名称。比如 int i = 10; 其中变量名称是i，变量中存储的值是10；int是数据类型，表示这个变量可以存储整型数值。在后面的教程中我们会继续详细讲解数据类型，这里我们就先讲解变量。

# Java中如何声明变量

使用下面的语法来声明变量：

data_type variable_name = value;

在Java中值是可选的，你可以先声明变量然后再对变量进行赋值。

比如这里num是变量，int是数据类型。我们先不管数据类型，后面章节将详细讲解数据类型，你只要知道数据类型使得变量num可以存储整型数值就好。你也可以阅读数据类型，但是我的建议是先阅读完本节内容再继续后面的。

int num;

同样的我们也可以在声明的同时对变量进行赋值，像下面这样：

char ch = 'A';
int number = 100;

或者也可以这样：

char ch;
int number;
...
ch = 'A';
number = 100;

# Java中变量的命名规范

  1. 变量名中不能包含空格，如：变量名中包含空格int num ber = 100;是错误的。
  2. 变量名可以使用像$和_这样的特殊字符作为开头。
  3. 相对标准的变量命名时是以小写字母开头，如int number;对于包含多个单词的变量名可以这样命名：int smallNumber; int bigNumber;（第二个单词首字母大写）。
  4. 在Java中变量名是大小写敏感的。

# Java中变量的种类。

Java中有三种不同的变量。

1)局部变量2)静态（类）变量3)实例变量

# 静态（类）变量

静态变量也被称为类变量，因为它关联的是类并且所有类的实例共享这个变量。比如，我可以创建一个类的3个实例对象来访问这个静态变量，它们是共享这个变量的，当某个对象改变了这个变量就会影响到其他对象。

# 静态变量的例子

public class StaticVarExample {
  public static String myClassVar="class or static variable";

  public static void main(String args[]) {
    StaticVarExample obj = new StaticVarExample();
    StaticVarExample obj2 = new StaticVarExample();
    StaticVarExample obj3 = new StaticVarExample();

    //All three will display "class or static variable"
    System.out.println(obj.myClassVar);
    System.out.println(obj2.myClassVar);
    System.out.println(obj3.myClassVar);

    //changing the value of static variable using obj2
    obj2.myClassVar = "Changed Text";

    //All three will display "Changed Text"
    System.out.println(obj.myClassVar);
    System.out.println(obj2.myClassVar);
    System.out.println(obj3.myClassVar);
  }
}

输出：

class or static variable
class or static variable
class or static variable
Changed Text
Changed Text
Changed Text

正如上面看到的不管哪个实例访问这个静态变量输出都是一样的。因此我们可以不用通过实例来访问静态变量，就像下面这样：

System.out.println(myClassVar);

注意：只有静态变量才可以这样访问。对于实例变量和局部变量是不行的。

# 实例变量

对于每个类的实例（对象）都有自己的一份实例变量。不像静态变量，每个实例有自己独有的一份实例变量的拷贝。当我们通过对象obj2改变实例变量的值，只有对象obj2中的值发生改变，其他对象中的值则不变。这也说名每个实例都有一份实例变量的拷贝。

# 实例变量的例子

public class InstanceVarExample {
  String myInstanceVar = "instance variable";

  public static void main(String args[]) {
    InstanceVarExample obj = new InstanceVarExample();
    InstanceVarExample obj2 = new InstanceVarExample();
    InstanceVarExample obj3 = new InstanceVarExample();

    System.out.println(obj.myInstanceVar);
    System.out.println(obj2.myInstanceVar);
    System.out.println(obj3.myInstanceVar);

    obj2.myInstanceVar = "Changed Text";

    System.out.println(obj.myInstanceVar);
    System.out.println(obj2.myInstanceVar);
    System.out.println(obj3.myInstanceVar);
  }
}

输出：

instance variable
instance variable
instance variable
instance variable
Changed Text
instance variable

# 局部变量

在类的方法中声明的变量就是局部变量。局部变量的作用域只限于方法中，也就意味着你不能在方法之外改变和访问它。

在下面的例子中，为了说明局部变量的作用域，我声明了一个实例变量和一个局部变量，两个变量的变量名相同。

# 局部变量的例子

public class VariableExample {
  // instance variable
  public String myVar = "instance variable";

  public void myMethod() {
    // local variable
    String myVar = "Inside Method";
    System.out.println(myVar);
  }
  public static void main(String args[]) {
    // Creating object
    VariableExample obj = new VariableExample();

    /* We are calling the method, that changes the
     * value of myVar. We are displaying myVar again after
     * the method call, to demonstrate that the local
     * variable scope is limited to the method itself.
     */
    System.out.println("Calling Method");
    obj.myMethod();
    System.out.println(obj.myVar);
  }
}

输出：

Calling Method
Inside Method
instance variable

如果我没有声明实例变量，只在方法中声明了局部变量。然后写上System.out.println(obj.myVar);这条语句将会抛出编译错误。因为在方法之外你不能访问局部变量。
