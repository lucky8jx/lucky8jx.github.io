# Java编程简介

1991年网景公司开发了JAVA，后来被甲骨文公司收购。Java是由James Gosling和Patrick Naughton一起编写的，它是一门简单的编程语言。在Java中编写，编译和调试一个程序都是相当简单的，并且利于开发模块化程序和编写可复用代码。

# Java技术

在我们开始学习Java之前，先熟悉下Java常用术语。

# Java虚拟机（JVM）

通常Java虚拟机被简写成JVM。在讨论JVM之前，我们先来看看程序执行的过程。首先是我们要编写一个程序，然后编译这个程序，最后运行这个程序。

  1. 由像你和我一样的程序员来编写这个程序。
  2. 由javac编译器编译这个程序。javac是Java开发工具（JDK）中自带的基础Java编译器。它可以把Java程序编译成Java字节码。
  3. 由JVM执行通过编译得到的字节码。这也就是程序的运行阶段。 现在我们知道了JVM的基本功能就是执行字节码，而字节码是通过编译器编译程序得来的。不同的操作系统有不同的JVM，编译出来的字节码却在不同的系统中都可以运行的。因此Java是跨平台语言。

# 字节码

如上所述，因为字节码是由JDK中javac编译器编译Java代码得来，所以它可以被JVM执行。编译器将会把字节码保存到.class后缀文件中。

# Java开发工具（JDK）

在讲解JVM和字节码时，我们提到JDK。现在我们就来聊聊它，就像它字面意思所体现的，JDK是一个Java开发工具，它完整的包含JRE（Java运行环境），编译器和很多像JavaDoc，Java调试器等等。 想要在我们的计算机中编写，编译和运行Java程序就必须安装JDK。

# Java运行环境（JRE）

JRE是JDK的一部分，也就是说JDK中自带有JRE的。只要在我们的系统中安装了JRE，我们就可以运行Java程序，但是我们不能对它进行编译。JRE中包含JVM，浏览器插件和支持小应用程序。如果只是像在我们计算机中运行Java程序，拿只要安装JRE就可以了。

以上就是容易使初学者感到迷惑的基础Java术语。可以查看完整的Java术语表，这是相应的连接https://docs.oracle.com/javase/tutorial/information/glossary.html

# Java主要特性

# Java是一门跨平台的语言

编译器（javac）可以把原始代码（.java后缀的文件）编译成字节码（.class后缀的文件）。正如上面提到的，JVM可以运行编译得到的字节码。字节码可以在所有操作系统中运行，包括Windows,Linux和Mac Os等等。也就是说，在windows中编译好的程序可以在Linux中运行，反过来也是可以的。每种操作系统中有不一样的JVM，但是在所有操作系统中运行同一个字节码得到的结果却是一样的。所以Java是一门跨平台语言。

# Java是一门面向对象语言

面向对象编程是把程序当成对象集合来组织的一种方法，每个对象都表示类的一个实例。

面向对象编程的4个主要概念

  1. 抽象
  2. 封装
  3. 继承
  4. 多态

# 简单

因为Java没有像操作符重载，多重继承，指针和显示内存分配这样的复杂特性，所以Java被认为是一门简单的编程语言

# 健壮性

健壮意味着可靠。Java编程语言在开发中着重在前期就检查出可能出现的错误，Java编译器可以检查出其他编程语言不容易探测到的一些错误。垃圾回收机制，异常处理和自动分配内存这些主要特性使得Java变得健壮。

# 安全

在Java中我们没有指针，也不能访问超出数组边界的内容（如果你这样做，将会得到数组索引超出边界的异常）。这也就是为什么在Java中是不会出现像堆栈或缓存溢出这样的错误。

# 分布式

Java编程语言可以用来开发分布式应用程序。在Java中RMI（远程方法调用）和EJB（企业级JavaBean）都是用来开发分布式用程序的。简单的说就是Java程序可以部署在不同系统中并通过网络来通讯。Objects on one JVM (java virtual machine) can execute procedures on a remote JVM.

# 多线程

Java支持多线程。多线程式Java的一个特性，它应许多个线程并行运行程序来达到CPU最大利用率。

# 可扩展性

就像上面说的，在一台机器上编写的程序可以在另外一台机器上运行。平台无关的字节码可以放在任意平台中运行，这就使得Java代码有更好的可扩展性。
