# 如何编译和运行你的第一个Java程序

在这篇教程里，你将了解到如何去写，编译和运行一个Java程序。这里也会讲解一些java的语法，编码规范和几种运行java程序的方法。

# 简单的Java程序：

public class FirstJavaProgram {
  public static void main (String[] args) {
    System.out.println("This is my first program in java");
  }//End of main
}//End of FirstJavaProgram Class

输出： This is my first program in java

# 如何编译和运行上面的程序

前提： 你需要在你的计算机中安装java。你可以在这里获取java。

第一步： 打开文本编辑器，像windows中的记事本和Mac中的文本编辑器。复制上面的程序然后粘贴到文本编辑器中。

你也可以使用IDE（集成开发环境）来运行java程序，如Eclipse。我们会在后面的教程中讲解这部分内容。为了简单起见，在这里我们只用文本编辑器和命令行（终端）。

第二步： 保存成FirstJavaProgram.java文件。你可能会想为什么文件名要命名成FirstJavaProgram，因为我们必须要保持文件名和public类的类名一致。在我们的程序中，public类的类名是FirstJavaProgram,因此把文件保存成FirstJavaProgram.java。

第三步： 在这里我们将编译程序。首先打开windows的命令行（cmd），如果是苹果系统则打开终端。

输入下面命令然后按下回车键来编译程序。

javac FirstJavaProgram.java

当你编译程序时可能会碰到这个错误：“javac'is not recognized as an internal or external command, operable program or batch file."如果在你的系统中没有设置java路径就会出现这个错误

如果你碰到这个问题，那么在编译之前先设置好路径。

在Windows中设置路径：

打开命令行（cmd），在java的安装目录中找到bin目录，拷贝这个完整路径然后在命令行中输入如下命令。

set path=C:\Program Files\Java\jdk1.8.0_121\bin

注意： 你的jdk版本可能和这个不一样。在我的系统中我安装的java版本是1.8.0_121，所以这边设置路径也用的是这个。

在苹果系统中设置路径

打开终端，输入下面的命令然后按下回车。

export JAVA_HOME=/Library/Java/Home

然后再输入下面的命令来确认是否设置成功。

echo $JAVA_HOME

上面设置的是临时路径，也就是说当你关掉命令行或者终端，设置会失效，下次要用时必须重新设置。永久的路径设置将再后面的教程教程中讲解。

第四步： 编译.java后缀文件会得到.class后缀文件（字节码）。现在我们就可以运行这个程序了。输入下面的命令并回车来运行这个程序：

java FirstJavaProgram

需要注意的是，再运行程序的时候不能在文件名后面加.java扩展名。

# 近距离探索我们的第一个Java程序

现在我们已经知道如何运行java程序，让我们更深入的了解下之前的代码。

public class FirstJavaProgram {

这是我们程序的第一行。每个java应用至少要有一个class关键字加类名的定义。这个关键字是不能改变的，类名却是比较随意的。

我通过使用公共类型访问修饰符使得这个类变的公开。我将分几节来讲解访问修饰符，现在你需要了解的是一个java文件中可以有任意多个类，但是只能有一个公开类，并且这个公开类名要和文件名一致。

public static void main(String[] args) {

这是我们程序中的第二行，为了更好的理解它，把它分解开来：

public： 这使得main这个方法变得公开，也就说是我们在这个类外面可以调用这个方法。

static： 这表示我们不需要创建一个实例来调用静态方法。类本身就可以调用它。

void： 这表示main方法不返回任何东西。

main： 这是方法名称。这是JVM运行你的程序的入口方法。

(String[] args)： 用来表示从命令行作为字符串传进来的一些参数。我们会有几节专门讲解这块内容。

System.out.println("This is my first program in java");

这个方法将在控制台打印出双引号中的内容并换行。
